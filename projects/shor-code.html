Shor Code Quantum Error Correction

About the Project

The Shor code is the first quantum error correction code that protects against both bit-flip and phase-flip errors. It encodes 1 logical qubit into 9 physical qubits, using 3 blocks of 3 qubits each.

How It Works

  • Encoding: 1 logical qubit → 9 physical qubits
  • Error injection: X (bit-flip), Z (phase-flip), Y (both)
  • Syndrome measurement: 2 bits for phase errors, 6 bits for bit-flip errors
  • Correction: Automatic correction based on measured syndromes
  • Decoding: Recovers the original logical state

Technical Details

  • Backend: FastAPI, Qiskit, Python 3.13
  • Frontend: Next.js, React, TypeScript
  • Deployment: Render (backend), Vercel (frontend)
  • Circuit size: 93 operations
  • Error types: X, Z, Y, none

Test Results

  • ✅ X error (bit-flip) on qubit 0 → syndrome: 100000 → corrected
  • ✅ Z error (phase-flip) on qubit 0 → syndrome: 10 → corrected
  • ✅ Y error (both) on qubit 0 → both syndromes → corrected
  • ✅ 100% correction rate in 1024 shots
