About the Project

Grover's algorithm provides a quadratic speedup for searching unsorted databases. This implementation searches for specific DNA sequences in a large DNA database, demonstrating quantum advantage over classical search.

How It Works

  • Superposition: Creates equal probability for all N items
  • Oracle: Marks target DNA sequences with a phase flip
  • Diffusion: Amplifies amplitude of marked states
  • Iterations: Optimal number = π/4 × √(N/M)
  • Measurement: Collapses to target positions with high probability

Technical Details

  • Backend: FastAPI, Qiskit, Python 3.13
  • Frontend: Next.js, React
  • Deployment: Render (backend), Vercel (frontend)
  • Complexity: O(√N) quantum vs O(N) classical
  • DNA Database: Custom .txt file upload
